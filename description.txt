I want to create a Chrome extension, it will be a private extension as I do not publish it online, so there should not be any strict restrction that I need to follow because I can load it from a local file. To summarize, the extension will do the following: when I'm on an amazon review page (by looking at the product URL), it will notify me on the taskbar that I can generate a review for this. and a write review button should appear somewhere. When I click on the write review button and select how many star on the extension, it should use other website that is already open, automate the task by using the finding the elements, send in the title, star etc. Then when the review is generated on a third party site website, copy that and paste it back into amazon review and select the correct number of stars, at this point do not submit, but wait for user to actually click on submit button on amazon review site. No sigin needed for amazon or any other AI site, we assume that user already logged in for those site.

Additional information:
- When user is on a review page with this url pattern : https://www.amazon.com/review/review-your-purchases/?asin=B01N2ZK39H, this is when the extension can be triggered
- Notice that there is an asin= code, this is the product asin
- With the asin we can go to this url https://www.amazon.com/dp/<asin> this is where we can extract the information of the product, like product title and images

Other requirements:
- In the extension, for now let's only let the user choose 2 options: Generate with Gemini and Generate with WordHero.ai, we will add more service later
- Keep the code in module, and in separated files for easy to maintain and fix
- For each of the option like Gemini or WordHero, keep the module to locate the element, submit text etc and other functions related to each of the service in a separated file
- Provide a user friendly UI/UX for the extension
- Write simple code and understandable code
- Have a button to turn off the extension so we don't waste resources and RAM
